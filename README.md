💰 Wage System
📌 About the Project

Wage System is a simple program that helps calculate the total wage of a labour/worker based on their working time.
In this program, the user can set the hourly rate as per requirement, then enter the check-in time and check-out time to automatically calculate the total working hours and the final wage.

✅ Features

Set custom wage rate per hour

Enter Check-In Time and Check-Out Time

Calculates total working time

Calculates total wage amount

Easy to use and fast output

⚙️ How It Works

User enters the hourly wage rate

User enters the check-in time

User enters the check-out time

Program calculates:

Total time worked

Total wage = (Total Hours Worked × Hourly Rate)

🧾 Example

Hourly Rate: 100 per hour
Check-In: 09:00
Check-Out: 17:00

Total Work Time: 8 hours
Total Wage: 800

🎯 Purpose

This project is useful for:

Daily wage workers calculation

Labour attendance & wage tracking

Small businesses and contractors

🛠️ Requirements

Any system that can run the program

Basic input of time and hourly rate
